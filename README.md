Автор:Даниил Лукин
# Описание проекта
Этот репозиторий содержит код для автоматического выделения снежно-ледового покрова Эльбрусского ледникового комплекса по спутниковым снимкам из Google Earth за период 2011–2020 гг. с использованием библиотеки компьютерного зрения OpenCV.
# Методология
В основе работы лежит метод пороговой обработки изображений:
1. Загрузка данных: Спутниковые снимки (GeoTIFF/JPEG) за декабрь каждого года получены из открытого сервиса Google Earth Pro.
2. Предобработка: Приведение снимков к единому разрешению (24 м/пиксель).
3. Бинаризация: Преобразование изображения в черно-белое с порогом `200`. Пиксели ярче порога (снег/лед) становятся белыми, остальные — черными.
4. Морфологическая обработка: Применение операций эрозии и дилатации для удаления шума и сглаживания границ.
5. Поиск контуров: Алгоритм находит все замкнутые контуры на бинарной маске.
6. Фильтрация: Удаляются контуры площадью менее 3000 пикселей (это отсеивает мелкие облака и снежные пятна).
7. Расчет площади: Площадь в пикселях переводится в км² по формуле: `S(км²) = (S(px) * (24 м/px)²) / 1 000 000`.
